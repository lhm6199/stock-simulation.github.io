# stock-simulation
ossp final project

게임화면 구현상황

npm install jquery --save

npm install apexcharts --save

해결한 것
- stock-table 행 클릭 시 해당 차트를 화면에 표시
- 시간이 흐르면서 주가가 변동되게 함

해결해야할 것
- 현재 시간 수정
- 차트 위 정보 표시
- 매수 / 매도 기능 구현
- 매수: user-table에 해당 주식 추가
- 매도: 해당 주식을 다 매도했을 경우 user-table에서 목록 삭제

추가 내용
- 시작 페이지 작성
- 이름 입력 시 다음 화면으로 전환
- 게임 화면에 이름 출력

시작페이지에 추가할 내용
- 체크박스 & 게임플레이 방법 안내
